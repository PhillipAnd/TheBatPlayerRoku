Configure The Bat Player

Using this web site you can customize the settings available on The Bat Player.

The first thing you'll want to do is add the stations you want to listen to. This is done via the Stations page. If you don't know where to start you can search and experiment with new ones.

If you have a Philips Hue lighting system you can set up your bulbs to match the look and feel of what's on the screen on the Lighting page. Pick and choose what bulbs you'd like to use and select what's the minimum and maximum brightness that you prefer.

Link The Bat Player with your Rdio account and add songs to a special playlist as you hear them. A great way to have on-demand access to the songs you discovered later.

And lastly if you're a hardcore Last.FM user like I am, you can configure up to two different accounts to scrobble the music you listen to on The Bat Player on the Last.FM configuration. Why two? You might have two people in the same household who both use Last.FM. Using your remote you can decide what accounts scrobble when.
