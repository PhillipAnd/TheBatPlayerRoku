Configure The Bat Player

Using this web site you can customize the settings available on The Bat Player.

The first thing you'll want to do is add the stations you want to listen to. This is done via the Stations page. If you don't know where to start you can search and experiment with new ones.

If you have a Philips Hue lighting system you can set up your bulbs to match the look and feel of what's on the screen on the Lighting page. Pick and choose what bulbs you'd like to use and select what's the minimum and maximum brightness that you prefer.

Link The Bat Player with your Rdio account and add songs to a special playlist as you hear them. A great way to have on-demand access to the songs you discovered later.

And lastly if you're a hardcore Last.FM user like I am, you can configure up to two different accounts to scrobble the music you listen to on The Bat Player on the Last.FM configuration. Why two? You might have two people in the same household who both use Last.FM. Using your remote you can decide what accounts scrobble when.

Release notes

1.4.7

  • Fixed the audio player problem that Roku 2 users saw with Roku OS 7.
  • Added some personality to the UI, or what I like to call "grunge mode".
  • Re-selecting the currently playing station will not interrupt the audio.
  • Fix for LastFM account selection that wouldn't let you select only the 2nd of two accounts.
  • Better, more accurate, more vibrant colors when using Hue lights.
  • Hopefully fixed the issue where launching the channel can take longer than it should.
  • Better UI performance for older devices.

1.4.6

  • You can now disable Last.FM scrobbling after you enabled it.
  • Add Station Wizard will warn you if you add a station that has the same name.
  • If you have a long station name it'll now get truncated on the Now Playing screen with an ellipsis.
  • As an experiment a Community page has been added to the web interface as a way to chat with other internet radio listeners and send questions and comments about The Bat Player.
  • The Bat Player is live on the Roku Channel Store now!

1.4.4

  • You can now exit from the station loading screen if you're waiting.
  • But if station metadata isn't available it will eventually give up instead of waiting there.
  • Some UI cleanup and performance tweaks.

1.4.0

  • More international/unicode characters should be supported.
  • UI Tweaks.
  • Changes to make images download faster.
  • Misc other performance enhancements and bug fixes.
  • More "Now Playing" information for stations when selecting stations and "On Other Stations" marquee.

1.3.0

  • New Add Station Wizard using the web config.
  • Support for AAC stations.
  • Improved speed fetching station data and resources.

As always if you have specific station you can't get to play feel free to reach out to me at gabek@real-ity.com and let me know the details of the station you're trying to add.
