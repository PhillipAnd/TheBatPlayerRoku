Edit

Hints:

  • Edit the following details to your liking, as it's how it'll be displayed on The Bat Player.
  • Shorten the station name if it looks a bit long.
  • The provider is the name of the group who creates the Station. It's optional.
  • If you don't know the format of the station, it's probably MP3. But some stations give the option to stream using the AAC format.
  • We try to find a suitable image for the station, but you should customize it. Find a nice image to represent the station. It can be anything you like.
  • You can come back to the Stations page at any time to update these values.
